# Digital Life Aquarium

OS上で動作する全てのプロセスを「デジタル生命体」として可視化するインタラクティブアート作品。

## 概要
コンピュータ内部のプロセスの生成、消滅、リソース消費を、水族館を泳ぐ生命体の生態として美しく表現します。

## インストール
```bash
pip install -r requirements.txt
```

## 実行
```bash
python main.py
```

## 開発スケジュール
- Week 1: 基盤構築と生命の誕生
- Week 2: 表現力の深化と完成

## 機能
- リアルタイムプロセスモニタリング
- プロセス情報の生命体としての可視化
- インタラクティブなUI
- fork/exec検知とアニメーション表現
